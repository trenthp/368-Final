The Brief

This project was made to be submitted to Eastern Washington University as an option to chaning their main student portal page. It is meant to give a quick one-stop-shop for all needed links, school news, student snapshots, etc. I built this using rapid prototyping techniques and built off of a persona.

The Approach

My approach starts with sketches and lo-fi builds. In this case I am strictly on the design side. I am team lead and working with the client and developers to create this app. We moved from sketches to wireframes to static mockups. We used Invision App to help build out interactive prototypes from our different iterations of prototypes.

The Result

The finished product (mainly dealing with the UI) went in a little different direction than what you see here. The app was built with ease of the user in mind. There is a main map page which allows the user to see where they are located in relation to all of the points of the hunt.
